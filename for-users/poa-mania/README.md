# POA Mania

A no-loss lottery where POA hodlers can win an extra slice of POA Pie ü•ß. Emission fund rewards are distributed to the winners every round. Featuring:

* üèÜ 3 winners per round. Rounds close daily.
* üéÜ Each round, 15% of prize pool is added to an ongoing jackpot. There is a 1% chance of winning the jackpot per round.
* üí∞ Prizes based on emission funds independent of the amount in the pool.
* ‚è≥ Close a round, win POA. 
* üé≤ On-chain, decentralized randomness decides the winners.

## üé≤ POA Mania 

POA Mania is currently available on POA core and the Sokol testnet. We are using the new RANDAO-based randomness contracts to support this decentralized, no-loss, on-chain lottery.
